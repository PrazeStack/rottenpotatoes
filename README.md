# RottenPotatoes

Simple Rails 4.2 demo app for the assignment.

Run locally:

```bash
# use Ruby 2.6.4 via rbenv
cd ~/rottenpotatoes
bundle _1.17.3_ install
bundle exec rails server -b 127.0.0.1 -p 3000
# open http://127.0.0.1:3000/
```

Notes: sqlite3 pinned to ~> 1.3.0 for Rails 4.2 compatibility.
